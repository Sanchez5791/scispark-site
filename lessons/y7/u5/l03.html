Living vs Non-Living — Judge with the Checklist

活的 vs 非活的 — 用清单判断
🧠 口诀金句 · Memory Hook
Do not ask only “does it move?” Ask “does it pass MRS GREN?”
不只问“会不会动”，要问“有没有过 MRS GREN”。 SLIDE L1 — MRS GREN recap (required each lesson L03-L08) EN: MRS GREN = Movement / Respiration / Sensitivity / Growth / Reproduction / Excretion / Nutrition. The 7 characteristics of living things. ZH: MRS GREN = 运动 / 呼吸 / 应激 / 生长 / 繁殖 / 排泄 / 营养。生物的 7 个特征。
⚖️ 考试 vs 真实 · Exam vs Reality
📝 考试这样写 · In the exam
  • In the exam, judge living things with MRS GREN.
  • 考试用 MRS GREN 判断活不活。
🔬 真实科学 · Real science
  • Some cases are messy, so scientists use evidence and definitions.
  • 真实世界有模糊例子，科学家看证据和定义。
MRS GREN recap (required each lesson L03-L08)
  • MRS GREN = Movement / Respiration / Sensitivity / Growth / Reproduction / Excretion / Nutrition.
  • The 7 characteristics of living things.
  • MRS GREN is the base of Y7 biology — every later biology unit uses it.
  • MRS GREN = 运动 / 呼吸 / 应激 / 生长 / 繁殖 / 排泄 / 营养。生物的 7 个特征。
  • MRS GREN 是 Y7 生物的基础，后续每个生物单元都会用到。

The 7-row Living Checklist (REQUIRED visual; conceptual model)
  • The Living Checklist.
  • Check a thing against all 7 rows.
  • Tick if it shows the characteristic, cross if it does not.
  • A thing is living only if it can show all 7 over its life.
  • Movement / 运动 [ tick / cross ]
  • Respiration / 呼吸 [ tick / cross ]
  • Sensitivity / 应激 [ tick / cross ]
  • Growth / 生长 [ tick / cross ]
  • Reproduction / 繁殖 [ tick / cross ]
  • Excretion / 排泄 [ tick / cross ]
  • Nutrition / 营养 [ tick / cross ]
  • 生命检查表。把一个东西对照 7 行检查。有这个特征打勾，没有打叉。
  • CHECKLIST_7_ROWS (list exactly, consistent visual across lessons):

MISCONCEPTION RED CARD (>=1 required)
  • WRONG: "If a thing moves, it is alive." A car moves but is not alive.
  • RIGHT: A thing is alive only if it shows all 7 characteristics.
  • 错：「会动就是活的。」 汽车会动但不是活的。
  • 对：要展现全部 7 个特征才算活的。
EXAM-vs-REAL-SCIENCE box (HIGH risk -> required)
🎯 考官要什么 · Command Words
  • State = give a short word/phrase · 短答, 直接给词
  • Describe = say what / how · 讲发生什么、怎么样
  • Explain = give a reason (fact + because) · 讲为什么 (事实+原因配对)

🔗 A robot is like a battery toy: it can act alive, but it does not run its own life processes. 机器人像电池玩具：动作像活的，但没有自己的生命过程。

Q06State / Decide Question: A robot toy can move across the floor, flash lights when it is touched, and switch itself off when its battery runs low. (a) Write down one characteristic of living organisms the robot seems to show. [1] (b) Decide whether the robot is living or non-living, and give one reason using the checklist. [2]3 marks
✅ 标准答案 · Answer
  • (a) movement
  • (a) 移动
可接受 · Accept
  • (a) movement OR sensitivity (responds to touch)
  • (b) non-living.
  • it cannot grow, reproduce, respire, excrete or take in nutrients on its own, so it fails the seven-sign checklist.
  • (a) 运动 OR 感应 (对触摸作出反应)
  • (b) 非活的。
  • it cannot 生长, 繁殖, 呼吸作用, 排泄 or 摄取营养 on its own, so it fails the 七个生命特征清单。
👍 Good — you judged with the full checklist.

Q08Explain Question: Fire can spread, give out heat, use up oxygen, and get bigger. Explain why fire is non-living, using two characteristics from the checklist that fire does not truly show. [2]2 marks
✅ 标准答案 · Answer
  • fire does not reproduce (cannot make more organisms of the same kind)
  • fire 不 繁殖 (不能 make more 生物 of the same kind)
可接受 · Accept
  • fire does not reproduce (cannot make more organisms of the same kind)
  • fire does not show nutrition / is not made of cells / does not excrete.
  • so it fails the checklist.
  • fire does not 繁殖 (cannot make more 生物 of the 同类)
  • fire does not show 营养 / is 不是由细胞构成 / 不排泄。
  • so it fails the 清单。
👍 Good — you judged with the full checklist.
