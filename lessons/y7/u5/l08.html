Living judge + virus borderline recap
  • Use the 7-row checklist to judge.
  • A car moves and uses fuel but fails most rows -> not living.
  • A virus is the hard case: it can reproduce (only inside a host) but has no independent nutrition, no growth, no respiration of its own -> it sits on the borderline.
  • 用 7 行清单来判断。一辆车会动、会用燃料,但大多数行都不符合 -> 不是活的。
  • 病毒是难判的个案:它能繁殖(只能在宿主体内),但没有独立营养、不生长、自己不呼吸 -> 它处在生命的边界上。

Species + classification + key recap, with Misconception RED CARD
  • Species = a group whose members 能交配 and make 可育后代 (a mule is not fertile, so horse and donkey are different species).
  • Bigger grouping uses the 7 levels: Kingdom -> Phylum -> Class -> Order -> Family -> Genus -> Species.
  • A dichotomous key is the hands-on tool: answer yes/no questions until you reach one name.
  • [MISCONCEPTION RED CARD] RED CARD EN: "Wrong idea: 'A key gives the right answer for any animal.' Correct: a key only works for the set of organisms it was built for.
  • An organism not in that set cannot be identified by that key." RED CARD ZH: "错误想法:'检索表对任何动物都能给出正确答案。' 正确:检索表只对它当初建立时所针对的那组生物有效。不在那组里的生物,这个表认不出来。"
  • 物种 = 彼此交配并产生有繁殖能力后代的一群(骡子不能繁殖,所以马和驴是不同物种)。
  • 更大的分组用 7 个层级:界 -> 门 -> 纲 -> 目 -> 科 -> 属 -> 种。
  • 二歧检索表是实战工具:不断回答 是/否 的问题,直到得到一个名字。

Q02Explain1 marks
A student says "A car is alive because it moves and uses fuel." Explain why the car is NOT living. Use the 7-row checklist in your answer.
💡 Tip Moving and using fuel are only 2 rows; check the other rows (growth, reproduction, etc.).
会动、会用燃料只是 2 行;检查其他行(生长、繁殖等)。
✅ 标准答案 · Answer
  • car fails growth / reproduction / sensitivity / excretion / nutrition (makes own food) — only mimics movement + energy use
  • car fails 生长 / 繁殖 / 感应 / 排泄 / 营养 (makes own food) — only mimics 移动 + energy use
可接受 · Accept
  • car fails growth / reproduction / sensitivity / excretion / nutrition (makes own food) — only mimics movement + energy use.
  • needs most/all 7.
  • car fails 生长 / 繁殖 / 感应 / 排泄 / 营养 (makes own food) — only mimics 运动 + energy use。
  • needs most/all 7。
👍 Good — you connected the answer to the right Unit 5 idea.
Q03Describe1 marks
Describe why a virus is called a "borderline case" between living and non-living. Give one life characteristic it has and one it lacks.
💡 Tip Think about what a virus can only do inside a host, and what it cannot do alone.
想想病毒只能在宿主体内做什么,以及它自己单独不能做什么。
✅ 标准答案 · Answer
  • HAS reproduction (only inside a host)
  • HAS 繁殖 (only inside a 宿主)
可接受 · Accept
  • HAS reproduction (only inside a host)
  • LACKS independent nutrition / growth / respiration / movement — any valid one each side.
  • HAS 繁殖 (只在宿主内)
  • LACKS independent 营养 / 生长 / 呼吸作用 / 运动 — any valid one each side。
👍 Good — you connected the answer to the right Unit 5 idea.

Q04State1 marks
A horse and a donkey can breed, but their 后代 (a mule) cannot have babies. State whether a horse and a donkey are the same species or different species, and give the reason.
✅ 标准答案 · Answer
  • different species
  • 不同物种
可接受 · Accept
  • different species.
  • because the offspring (mule) is not fertile / cannot reproduce, so they do not meet the fertile-offspring definition.
  • 不同物种。
  • 因为后代（骡子）不可育/不能繁殖，所以不符合可育后代的定义。
