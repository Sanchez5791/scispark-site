Complete Food Web: Add Back the Decomposer

完整食物网：把分解者加回来
Every Web We Drew Was Missing One Thing我们画过的每张网都少了一样东西
  • Look back at the food webs from earlier lessons.
  • Grass, rabbit, fox, owl — all alive.
  • But every living thing dies one day.
  • Where do the dead bodies go?
  • One type of living thing was never on the web.
  • 回头看前几课画的食物网。
  • 草、兔、狐、猫头鹰 — 全是活的。
  • 但每个生物总有一天会死。
  • 那些死掉的身体去了哪里?有一类生物从来没出现在网上。
😄
A food web with no decomposer is like a classroom with no cleaner: the mess does not magically vanish.
没有分解者的食物网，就像没有清洁工的课室：垃圾不会自己消失。

Who Does This for a Living?谁靠这个吃饭?
  • Field ecologist / conservation biologist.
  • They study how all the organisms in one place connect — including the decomposers most people forget.
  • A web is only "real" when it is complete.
  • 野外生态学家 / 保育生物学家。
  • 他们研究一个地方里所有生物怎么连在一起 — 包括大多数人忘掉的分解者。
  • 一张网只有完整时才算"真实"。

Learn · Define

Complete Food Web: Add Back the Decomposer

完整食物网：把分解者加回来
What Makes a Food Web "Complete"什么让食物网"完整"
  • A complete food web shows producers, consumers AND decomposers, with every dead organism and its waste linked to the decomposers.
  • 完整食物网要画出生产者、消费者，还有分解者，并把每个死掉的生物和它的废物都连到分解者。
Producers make food (plants)
生产者造食物 (植物)
Consumers eat other living things
消费者吃别的活生物
Decomposers take in dead matter + waste
分解者吸收死物 + 废物
🧠
Complete web = P + C + D: Producers, Consumers, Decomposers.
完整网 = 生 + 消 + 分：生产者、消费者、分解者。
📌 Without decomposers, the web is incomplete.
🖼️

Decomposer Arrows Are Dashed — and Point INTO the Decomposer分解者的箭头是虚线 — 而且指向分解者
  • Solid arrows = "is eaten by", from prey to predator.
  • Dashed arrows = dead matter and waste going to the decomposer.
  • The dashed arrow points TO the decomposer.
  • 实线箭头 = "被吃"，从被吃的指向吃的。
  • 虚线箭头 = 死物和废物流向分解者。
  • 虚线箭头指向分解者。
Solid arrow: prey → predator (energy flow)
实线: 被吃的 → 吃的 (能量流)
Dashed arrow: dead body / waste → decomposer
虚线: 死物/废物 → 分解者
Decomposers do NOT eat living prey
分解者不吃活的猎物
📌 Dashed lines show the decomposer receives "dead/waste", not "live prey".

Your Turn — Try These

练一练
Q01 L1 1 mark
In a complete food web, which arrow is drawn DASHED?
在完整食物网里，哪一条箭头要画成虚线?
A) grass → rabbit
B) rabbit → fox
C) dead fox → decomposer
D) fox → owl
💡 Hint 1
Dashed arrows are only for dead matter or waste going to the decomposer, not for one animal eating another.
虚线只用在死物或废物流向分解者，不是一只动物吃另一只。
✅ 答案 Answer
C) dead fox → decomposer.
C) 死狐狸 → 分解者。
可接受 Accept:
  • C; dead fox to decomposer; dead matter/waste to decomposer.
Q02 L2 3 marks
This food web has grass, rabbit, fox and a decomposer. Draw the dashed arrows needed to make the web complete.
这张食物网有草、兔、狐和一个分解者。画出需要的虚线箭头，让这张网变完整。
🖼️

💡 Hint 1
Every living thing eventually dies. Each one's dead body must connect to the decomposer.
每个生物最后都会死。每个的死物都要连到分解者。
✅ 答案 Answer
A dashed arrow from each organism (grass, rabbit, fox) and from dead matter/waste pointing INTO the decomposer.
从每个生物 (草、兔、狐) 以及死物/废物各拉一条虚线，指向分解者。
可接受 Accept:
  • dashed arrow from grass/dead plant matter into decomposer
  • dashed arrow from rabbit/dead rabbit/waste into decomposer
  • dashed arrow from fox/dead fox/waste into decomposer
  • all decomposer arrows point into the decomposer
  • dead matter/waste box points into decomposer if used

Test Yourself

独立测试
Q03 L2 Explain 2 marks
Explain why a decomposer's arrows are drawn dashed and not solid.
解释为什么分解者的箭头画成虚线而不是实线。
✅ 答案 Answer
Because the decomposer takes in dead matter and waste, not living prey; dashed shows it is not "eating" a live organism.
因为分解者吸收的是死物和废物，不是活的猎物;虚线表示它不是在"吃"活生物。
可接受 Accept:
  • dead/waste not live prey
  • dashed distinguishes decomposer flow
  • dashed arrow shows decomposer gets dead matter/waste, not live food
Q04 L2 State 2 marks
A student draws a food web with grass, rabbit and fox but no decomposer. Why is this web NOT complete?
一个学生画了草、兔、狐的食物网，但没有分解者。为什么这张网不算完整?
✅ 答案 Answer
A complete web must show what happens to dead organisms and waste; without a decomposer the dead matter has nowhere to go.
完整的网必须显示死掉的生物和废物去哪里;没有分解者，死物就无处可去。
可接受 Accept:
  • missing decomposer
  • dead matter/waste not shown
  • mentions dead organisms/waste need a path to a decomposer
Q05 L3 Name 3 marks
In one complete food web, name TWO things that send a dashed arrow to the decomposer, and say where the decomposer's "food" comes from.
在一张完整食物网里，说出两样会拉虚线指向分解者的东西，并说明分解者的"食物"来自哪里。
✅ 答案 Answer
Two from: dead plants, dead animals, animal waste. The decomposer's food comes from dead matter and waste, not from living organisms.
两样取自: 死植物、死动物、动物废物。分解者的食物来自死物和废物，不是活生物。
可接受 Accept:
  • any two dead plant/dead animal/waste
  • food from dead matter/waste not living prey
  • dead plants, dead animals, or waste named as decomposer food
Q06 L1 State 2 marks
State what type of arrow is used to show energy going to a decomposer in a complete food web.
写出在完整食物网里, 表示能量流向分解者所用的是哪种箭头。
✅ 答案 Answer
A dashed (broken) arrow.
虚线(断开的)箭头。
可接受 Accept:
  • dashed arrow
  • broken arrow
  • dotted/dashed line accepted if it clearly means a broken arrow
Q07 L2 State 2 marks
In a complete food web, state which way a decomposer arrow points: into the decomposer or out of it.
在完整食物网里, 写出分解者箭头指向哪边: 指入还是指出分解者。
✅ 答案 Answer
Into the decomposer.
指入分解者。
可接受 Accept:
  • into the decomposer
  • towards decomposer
  • arrow points from dead matter/waste into the decomposer
Q08 L2 Describe 2 marks
Describe what is missing from a food web that shows only producers and consumers.
描述一个只显示生产者和消费者的食物网少了什么。
✅ 答案 Answer
It is missing the decomposers, which break down dead matter and recycle nutrients.
它少了分解者, 分解者分解死物并回收养分。
可接受 Accept:
  • missing decomposers
  • break down dead matter
  • decomposers recycle nutrients from dead organisms
Q09 L3 Explain 3 marks
Explain why a decomposer in a complete food web receives energy from every other organism.
解释为什么完整食物网里的分解者会从每个其他生物获得能量。
✅ 答案 Answer
Because every organism eventually dies or produces waste, and the decomposer breaks that down, so energy from all of them passes to the decomposer.
因为每个生物最终都会死亡或产生废物, 分解者分解这些, 所以所有生物的能量都传给分解者。
可接受 Accept:
  • every organism dies
  • every organism makes waste
  • decomposer breaks down their dead bodies/waste so energy passes to it
Q10 L1 State 2 marks
State why decomposers are sometimes called the recyclers of an ecosystem.
写出为什么分解者有时被称为生态系统的回收者。
✅ 答案 Answer
Because they break down dead matter and return nutrients to the soil for plants to reuse.
因为它们分解死物, 把养分送回土壤供植物再利用。
可接受 Accept:
  • break down dead matter
  • return nutrients
  • nutrients go back to the soil for plants to reuse
