Every measurement has some error — good scientists know which kind

每个测量都有误差 —— 好的科学家知道是哪一种
🌍 Real-world · 真实情境
• A pharmacist measuring a dose of medicine must know where errors come from.
• A scale that always reads high (systematic error) is dangerous in a different way from a hand that shakes (random error).
• Knowing the type of error tells a scientist how to fix it.
• Getting this wrong, in medicine or engineering, can cost lives.
• 量药剂量的药剂师必须知道误差从哪里来。
• 一台总是读高的秤(系统误差),跟一只发抖的手(随机误差), 危险的方式不一样。
• 知道误差的种类,科学家才知道怎么修正它。
• 在医药或工程里搞错这个,可能要人命。
📘
What this lesson is about
这一课讲什么
• No measurement is perfect; the question is what kind of error it has.
• 没有完美的测量;问题是它有哪一种误差。
• This lesson: systematic, random and gross errors, plus zero error and parallax error.
• 这一课:系统误差、随机误差、粗心误差,加上零误差和视差误差。

• 误差不是说你笨。
• 科学里的误差是读数可能和真值不一样。

测量很诚实,但工具有时候心情不好。

Errors in Measurement

学习重点
🔎
Systematic error
系统误差
• Affects all measurements in the same direction (always too high, or always too low).
• 影响所有测量,且方向相同(总是偏高,或总是偏低)。
• It is identifiable and fixable (e.g. re-zero or calibrate the instrument).
• 它是可识别、可修正的(例如重新归零或校准仪器)。
• It makes measurements less accurate.
• 它让测量变得不那么准确。
🧩 Analogy · 比喻
Systematic error is like a weighing scale that lies by the same amount every day.
系统误差像每天都多报同样重量的体重秤。
🧠 Mnemonic · 口诀
Systematic = same direction, same problem.
记法:系统误差=同方向,同毛病。
🖼️
IMAGE PENDING · 配图待出  [L05_Q00_IMG02]
Systematic error features (same direction / identifiable & fixable / less accurate).
⚖️
Random error and gross error
随机误差和粗心误差
• Random error: caused by fluctuations in variables; uncontrollable; affects the spread or causes anomalous results.
• Reduced by repeating and averaging.
• 随机误差:由变量的波动造成;不可控;影响数据的分散或造成异常结果。
• 用重复测量并取平均来减小。
• Gross error: caused by carelessness and accidents (e.g. misreading a scale, recording the wrong number).
• 粗心误差:由粗心和意外造成(例如读错刻度、记错数字)。
🔎 Plain · 白话
• Random error jumps around.
• Repeats and averages help smooth it out.
• 随机误差会跳来跳去。
• 重复和平均可以把它压平一点。
🧩 Analogy · 比喻
It is like throwing crumbs: each piece lands a bit differently.
像撒面包屑:每一粒落点都差一点。
🖼️

🧪
Zero error
零误差
• Zero error happens when an instrument does not read zero when it should.
• 零误差发生在仪器本该读 0 时却没有读 0。
• No zero error: reads 0 correctly at the start.
• 无零误差:开始时正确读 0。
• Positive zero error: reads above 0 when it should read 0.
• 正零误差:本该读 0 时读到大于 0。
• Negative zero error: reads below 0 when it should read 0.
• 负零误差:本该读 0 时读到小于 0。
• Zero error is a type of systematic error.
• 零误差是一种系统误差。
🔎 Plain · 白话
Zero error starts wrong before the measurement even begins.
零点误差是在还没量之前就已经歪了。
🖼️
IMAGE PENDING · 配图待出  [L05_Q00_IMG04]
Three scale close-ups: no zero error / positive zero error / negative zero error.
🧩
Parallax error
视差误差
• Parallax error happens when you read a scale from the wrong angle.
• 视差误差发生在你从错误的角度读刻度时。
• Correct: eye level with the scale, looking straight on.
• 正确:眼睛与刻度齐平,正对着看。
• Wrong: looking from above or below gives a reading that is too high or too low.
• 错误:从上方或下方看,会读到偏高或偏低的值。
• Read liquid levels at the bottom of the meniscus, at eye level.
• 读液面时,要在视线齐平处读弯月面的底部。
🧩 Analogy · 比喻
Parallax is the "wrong seat" problem: look from the side, and the reading shifts.
视差就是「坐错位」问题:从旁边看,读数会跑位。
🧠 Mnemonic · 口诀
Eye level, not sky level.
记法:眼平看,别天上看。

Repeating does not remove a systematic error
重复测量去不掉系统误差
🔎 Plain · 白话
Repeating can reduce random error, but it cannot cure a broken zero.
重复能减少随机误差,但治不好坏掉的零点。
😄 Humour · 幽默
If your ruler starts at 2 mm, repeating five times only gives you five confident wrong answers.
尺从 2 mm 开始,量五次只会得到五个很自信的错答案。
🚩 COMMON MISTAKE · 常见错误
✘ Students often think · 学生常以为
Students think repeating a measurement and averaging removes all errors.
学生以为重复测量并取平均能去掉所有误差。
✔ Correct · 正确
• Repeating and averaging reduces RANDOM error, because random errors scatter both ways.
• But a SYSTEMATIC error pushes every reading the same way, so averaging does not remove it — you must fix the cause (e.g. re-zero or calibrate).
• 重复并取平均能减小"随机"误差,因为随机误差向两边散开。
• 但"系统"误差把每个读数推向同一个方向,所以取平均去不掉它
• 你必须修正起因(例如重新归零或校准)。
🎯 What the examiner wants · 考官要什么
State = short answer · Describe = say what & how · Explain = give the reason (fact + because) · Define = use the exact term.
State=短答 · Describe=说什么、怎样 · Explain=讲原因(事实+because) · Define=用准确术语。

Q05 Explain · 3 marks · short_answer
A ruler is worn down so its scale starts at 2 mm instead of 0.
Explain what type of error this causes and why repeating the measurement
will not fix it.
一把尺子磨损了,刻度从 2 毫米开始而不是 0。
解释这会造成什么类型的误差,以及为什么重复测量无法修正它。
💬 Tip · 提示: Zero error is systematic; averaging will not remove it. 零误差是系统性的;取平均去不掉它。
🔑 Hint · 提示
Is the 2 mm offset the same every time, or does it scatter?
这 2 毫米的偏移每次都一样,还是会散开?
🔑 Hint 2 · 第二提示
Averaging only helps when errors go both ways.
取平均只在误差向两边散开时才有用。
✅ Show answer · 看答案
Answer · 答案

一个零误差,属于系统误差:每个读数都向同一个方向 偏移相同的 2 毫米。重复测量每次得到同样的偏移,所以取平均去不掉它 —— 你必须修正这个零点(例如从真正的零开始测量,或减去 2 毫米)。
Mark scheme · 评分: identifies zero/systematic error AND explains repeats keep the same offset; accept: A zero error, which is a systematic error: every reading is off by the same 2 mm in the same direction. Repeating gives ; accept also: gives the same scientific reason in different words.
指出零/系统误差,并解释重复测量保持同样偏移；接受：一个零误差,属于系统误差:每个读数都向同一个方向 偏移相同的 2 毫米。重复测量每次得到同样的偏移,所以取平均去不掉它 —— 你必须修正这个零点(例如从真正的零开始测量,或减去 2 毫米)。；也接受：用不同说法讲出同一个科学原因。
If wrong · 答错: The ruler’s zero is shifted before measuring begins. That makes every reading wrong by the same amount, so it is zero error and systematic error. 尺的零点在量之前已经偏了。每次读数都会偏同样的量,所以是零点误差,也是系统误差。

WRAP · 总结

Know your error, then you can fix it

认得误差,才能修正它
📌
Key points to keep
记住这几点
• Systematic: same direction every time; fix the cause (re-zero/calibrate).
• 系统:每次同一个方向;修正起因(重新归零/校准)。
• Random: scatters both ways; reduce by repeating and averaging.
• 随机:向两边散开;用重复取平均减小。
• Gross: carelessness/accident; avoid by careful work.
• 粗心:粗心/意外;靠仔细做避免。
• Zero error: instrument not reading 0 (a systematic error).
• 零误差:仪器没读 0(一种系统误差)。
• Parallax error: wrong viewing angle; read at eye level.
• 视差误差:视线角度错;在视线齐平处读。
