One change at a time — that is what makes a test fair

一次只改一样 —— 这才让测试公平
🌍 Real-world · 真实情境
• An agricultural scientist testing which fertiliser grows the tallest crops must give every plant the same water, light, and soil, and only change the fertiliser.
• If they changed two things at once, they would never know which one caused the result.
• Keeping a test fair is the core skill behind every reliable experiment.
• 测试哪种肥料能种出最高作物的农业科学家,必须给每株植物 一样的水、光、土,只改肥料。
• 如果一次改两样, 就永远不知道是哪一样造成了结果。
• 让测试保持公平,是每个可靠实验背后的核心本事。
📘
What this lesson is about
这一课讲什么
• Different questions need different types of investigation.
• 不同的问题需要不同种类的探究。
• A fair test changes only one thing and keeps the rest the same.
• 公平测试只改一样,其余保持不变。
• This lesson: types of investigation, fair vs biased, and the three kinds of variable.
• 这一课:探究种类、公平vs有偏、以及三种变量。

• 开始前:想一个你看过的实验。

🔎 Plain · 白话
A fair test is strict because it wants one clear cause, not a messy soup of causes.
公平测试很严格,因为它只想抓一个清楚原因,不要一锅乱汤。
🧠 Mnemonic · 口诀

口诀:改一个,量一个,其他守住。

Fair Testing and Variables

学习重点
🔎
Four types of investigation
四种探究
• Fair test: change one variable, keep all others the same.
• 公平测试:改变一个变量,其余全部保持不变。
• Making a model: physical model (large/small systems) or computer model.
• 建模型:实体模型(大型/小型系统)或电脑模型。
• Field study: investigating plants or animals in their natural habitat.
• 田野调查:在自然栖息地里研究植物或动物。
• Survey / regular observations: data over a long time, or many places.
• 调查 / 定期观察:长时间收集数据,或在很多地方收集。
🔎 Plain · 白话
• Different questions need different investigation types.
• Do not use one tool for every job.
• 不同问题要用不同探究方式。
• 不要拿一把工具打天下。
🧩 Analogy · 比喻
• It is like choosing shoes: running shoes, school shoes, slippers — same feet, different jobs.
• 像选鞋:跑鞋、校鞋、拖鞋。
• 脚一样,任务不一样。

⚖️
Fair test vs biased test
公平测试 vs 有偏测试
• Fair test: able to produce reliable AND valid results.
• 公平测试:能产生可靠而且有效的结果。
• Biased test: may be reliable to some extent, but NOT valid.
• 有偏测试:可能在某种程度上可靠,但不有效。
• 「有效」的意思是这个测试真的测到了它应该测的东西。
• "Valid" means the test really measures what it is supposed to.
🔎 Plain · 白话
Bias enters when another factor secretly pushes the result.
偏差就是有另一个因素偷偷推了结果。
😄 Humour · 幽默
• If you change water and light together, the plants can blame either one.
• Very convenient for confused plants.
• 如果水和光一起改,植物可以怪任何一个。
• 植物很方便,学生很惨。

🧪
Independent, dependent, and control variables
自变量、因变量、控制变量
• Independent variable: the one thing you CHANGE.
• 自变量:你改变的那一样东西。
• Dependent variable: the thing you MEASURE (the result).
• 因变量:你测量的东西(结果)。
• Control variables: the things you KEEP THE SAME so the test is fair.
• 控制变量:你保持不变的东西,让测试公平。
🧩 Analogy · 比喻
• Independent variable is the remote control.
• Dependent variable is what the screen does.
• 自变量像遥控器。
• 因变量像屏幕出现的变化。
🧠 Mnemonic · 口诀
I change the Independent. I measure the Dependent. Controls stay Constant.
记法:I 我改,D 我量,C 不动。
🧩
Worked example: growing seeds
例子:种种子
• Pot with soil -> put seeds of species A into soil -> water the soil -> leave for 10 days -> measure the height.
• 装土的花盆 -> 把 A 物种的种子放进土 -> 给土浇水 -> 放 10 天 -> 测量高度。
• If we test how water amount affects height:
• 如果我们测试"水量怎样影响高度":
• Change (independent): amount of water.
• 改变(自变量):水的量。
• Measure (dependent): height of the plant.
• 测量(因变量):植物的高度。
• Keep same (control): same soil, same seeds, same pot, same light.
• 保持不变(控制):一样的土、一样的种子、一样的盆、一样的光。
🔎 Plain · 白话
In the seed test, fertiliser is the thing you change; plant height is the thing you measure.
种子实验里,肥料是你改的东西;植物高度是你量的东西。

Changing more than one thing breaks the test
一次改超过一样,测试就坏了
🔎 Plain · 白话
• Two changes at once destroy the evidence.
• You cannot tell which change caused the result.
• 一次改两个,证据就坏了。
• 你分不出到底是谁造成结果。
🧩 Analogy · 比喻
If soup becomes salty after adding salt and soy sauce together, you cannot blame only one.
汤变咸,但你同时加盐和酱油,你不能只怪一个。
🚩 COMMON MISTAKE · 常见错误
✘ Students often think · 学生常以为
Students change two variables at once (e.g. more water AND more light) and still think the test is fair.
学生一次改两个变量(例如更多水"加"更多光), 还以为测试是公平的。
✔ Correct · 正确
• A fair test changes only ONE variable (the independent variable).
• If two things change, you cannot tell which one caused the result, so the test is not valid.
• 公平测试只改"一个"变量(自变量)。
• 如果改了两样,你就分不出是哪一样造成结果,所以测试不有效。
🎯 What the examiner wants · 考官要什么
State = short answer · Describe = say what & how · Explain = give the reason (fact + because) · Define = use the exact term.
State=短答 · Describe=说什么、怎样 · Explain=讲原因(事实+because) · Define=用准确术语。

Q05 Explain · 3 marks · short_answer
A student gives one plant more water AND more light than the
other, then says light made it grow taller. Explain why this conclusion
is not valid.
一位学生给一株植物比另一株更多水"加"更多光,
然后说是光让它长得更高。解释为什么这个结论不有效。
💬 Tip · 提示: Two changes mean you cannot tell which caused the result. 改两样意味着你分不出是哪样造成结果。
🔑 Hint · 提示
How many things were changed between the two plants?
两株植物之间改了几样东西?
🔑 Hint 2 · 第二提示
If two inputs change together, the result could be from either.
如果两个输入一起改,结果可能来自任一个。
✅ Show answer · 看答案
Answer · 答案
Two variables were changed (water and light), so you cannot tell which one caused the extra height. The test is not fair, so the conclusion is not valid.
改了两个变量(水和光),所以分不出是哪一个造成多长的高度。 测试不公平,所以结论不有效。
Mark scheme · 评分: any answer noting two variables changed -> cannot attribute the result; accept: Two variables were changed (water and light), so you cannot tell which one caused the extra height. The test is not fair; accept also: gives the same scientific reason in different words.
任何指出"改了两个变量 -> 无法归因结果"的答案；接受：改了两个变量(水和光),所以分不出是哪一个造成多长的高度。 测试不公平,所以结论不有效。；也接受：用不同说法讲出同一个科学原因。
If wrong · 答错: The problem is not that the plants grew. The problem is that water and light both changed, so the cause cannot be identified. 问题不是植物长高了。问题是水和光都变了,所以分不出真正原因。

Q09 State · 2 marks · short_answer
State why control variables must be kept the same in a fair test.
说出为什么公平测试里控制变量必须保持不变。
✅ Show answer · 看答案
Answer · 答案
So that only the independent variable can affect the result, making the test fair and the result valid.
这样只有自变量能影响结果,让测试公平、结果有效。
Mark scheme · 评分: any answer linking controls -> only the independent affects the result; accept: So that only the independent variable can affect the result, making the test fair and the result valid.; accept also: uses the correct variable names even if sentence grammar is weak.
任何把"控制变量 -> 只有自变量影响结果"连起来的答案；接受：这样只有自变量能影响结果,让测试公平、结果有效。；也接受：变量名称正确,就算句子语法弱也接受。
If wrong · 答错: Controls are kept the same so the result can be blamed on the independent variable only. If other factors change, the test becomes unfair. 控制变量保持相同,结果才可以只归因于自变量。其他因素也变,测试就不公平。
