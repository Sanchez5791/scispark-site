Unit Review and Integration
The whole-unit chain
  • Properties (physical vs chemical) -> acids and alkalis (pH / indicators / classification) -> gas tests (hydrogen, oxygen, carbon dioxide) -> metals, non-metals and alloys.
  • Each part connects to the next.
  • 性质 (物理 vs 化学) -> 酸 和 碱 (pH / 指示剂 / 分类) -> 气体 测试 (氢、氧、二氧化碳) -> 金属、非金属 和 合金。
  • 每 一 块 接 着 下 一 块。
Review: properties and materials
  • Materials are chosen by their properties: strong, ductile (can be drawn into wires), magnetic, sonorous, transparent.
  • Metals are usually strong, ductile and conduct;
  • non-metals usually are not.
  • 材料 是 按 它们 的 性质 来 选 的: 强、可 延展 (能 拉 成 线)、 有 磁性、能 发声、透明。
  • 金属 通常 强、可 延展、能 导电;
  • 非金属 通常 不 是。
🟥 红牌 · 常见错误 / Red Card
✘ WRONG 错
All metals are magnetic.
所有 金属 都 有 磁性。
✔ RIGHT 对
Only iron, nickel, cobalt (and steel, which contains iron) are magnetic. Copper, aluminium and gold are metals but are NOT magnetic. Magnetic is not the same as metallic.
只有 铁、镍、钴 (还有 含 铁 的 钢) 有 磁性。 铜、铝、金 是 金属, 但 没有 磁性。有 磁性 跟 是 金属 不 是 一 回事。
⚖️ 考试 vs 真实 / Exam vs Reality
  • In the exam, write that the pH scale goes from 1 to 14.
  • In real life, very strong acids can have a pH below 0, and very strong bases above 14.
  • The 1-14 scale is a simplified range for everyday substances.
  • 考试 时 写: pH 是 1 到 14。
  • 真实 中, 很 强 的 酸 pH 可 到 0 以下, 很 强 的 碱 可 到 14 以上。
  • 1-14 只 是 给 日常 物质 用 的 简化 范围。
🔗 比喻 / Analogy
  • The unit is like a detective case: each clue points to one material, gas, or substance type.
  • 整个单元像侦探案：每个线索指向一种材料、气体或物质类型。
🎯 考官要什么 / What the examiner wants
  • State = short answer.
  • Describe = say what and how.
  • Explain = say why (fact + reason).
  • Define = exact syllabus wording.
  • State = 简短作答。
  • Describe = 说出是什么、怎样。
  • Explain = 说出为什么(事实+原因)。
  • Define = 用大纲原话精确定义。
