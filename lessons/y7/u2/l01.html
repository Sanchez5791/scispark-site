⏰ 热身 / Warm-up
  • Pick one common object near you and ask: which property is useful here?
  • 看身边一个普通物品，问自己：这里真正有用的是哪个性质？
  • A materials engineer must choose the right material for a product.
  • A drinks can must be light and must not react with the drink inside.
  • A bridge cable must be strong and must not snap.
  • To choose well, the engineer compares the PROPERTIES of materials.
  • 材料工程师要为产品挑对的材料。
  • 饮料罐要轻,而且不能跟里面的饮料反应。
  • 桥的钢索要强,而且不能断。
  • 要挑得对,工程师就要比较材料的"性质"。
😄 A material is not a dating profile: “shiny” is not enough. We need the useful properties too. 材料不是交友档案：只写“很闪”不够，还要看真正有用的性质。

Physical vs Chemical Properties

物理性质与化学性质
What is a physical property
  • A physical property can be measured or observed WITHOUT changing the substance into a new substance.
  • Examples: melting point, boiling point, state at room temperature, density, hardness, whether it conducts electricity.
  • 物理性质是不把物质变成新物质就能测量或观察到的性质。
  • 例子:熔点、沸点、 室温下的状态、密度、硬度、能不能导电。
What is a chemical property
  • A chemical property describes how a substance reacts to form a NEW substance.
  • Examples: whether it burns, whether it reacts with acid, whether it corrodes.
  • 化学性质描述一个物质如何反应、生成新物质。
  • 例子:会不会燃烧、会不会跟 酸反应、会不会被腐蚀。
🟥 红牌 · 常见错误 / Red Card
✘ WRONG 错
If the shape or state changes, that is a chemical change.
形状或状态变了,就是化学变化。
✔ RIGHT 对
Changing shape or melting is a PHYSICAL change — no new substance is made. A chemical change makes a new substance.
改变形状或熔化是物理变化——没有生成新物质。化学变化会生成新物质。

⚖️ 考试 vs 真实 / Exam vs Reality
  • "In the exam, judge a chemical change by asking 'is a NEW substance made?'.
  • Do not rely on 'can it be reversed?' — that test alone can be wrong."
  • "考试里,判断化学变化要问'有没有生成新物质?'。
  • 不要只靠'能不能逆转', 单看这个会判错。

🔗 比喻 / Analogy
  • Think of wax melting like changing clothes: the look changes, but the person is still the same.
  • Burning wood is different: it makes ash and gas.
  • 蜡熔化像换衣服：样子变了，人还是同一个。
  • 木头燃烧不同：会变成灰和气体。

WRAP · 总结

Physical vs Chemical Properties

物理性质与化学性质
🔁 盖住回想 / Cover & recall
A physical property is measured without making a new substance (melting point, state, hardness).
A chemical property describes how a substance reacts to make a new substance (burning, reacting with acid). Engineers compare both to choose the right material.
📌 今天学了什么 / What we learned
  • A physical property is measured without making a new substance (melting point, state, hardness).
  • A chemical property describes how a substance reacts to make a new substance (burning, reacting with acid).
  • Engineers compare both to choose the right material.
  • 物理性质不生成新物质就能测量(熔点、状态、硬度)。
  • 化学性质描述物质如何反应、 生成新物质(燃烧、跟酸反应)。
  • 工程师比较这两类来挑对的材料。
🧠 口诀金句 / Memory hook
Memory hook: Physical = “no new stuff”; Chemical = “new stuff made”.
记法：物理 = 没有新东西；化学 = 生成新东西。
