Review: evidence and indicators
  • A scientific statement needs evidence.
  • To test if a liquid is an acid, use an indicator (e.g.
  • litmus, pH paper, Universal Indicator).
  • An acid turns litmus red.
  • 科学 说法 要 有 证据。
  • 要 测 一 个 液体 是 不 是 酸, 用 指示剂 (litmus / pH 纸 / Universal Indicator)。
  • 酸 让 litmus 变 红。
Review: hazard symbols
  • Hazard symbols warn about dangers of a substance: toxic, irritant / harmful, oxidising, corrosive, flammable, health hazard, hazardous to the environment.
  • Match the symbol to its meaning before handling.
  • 危险 标志 警告 一 种 物质 的 危险: 有毒、刺激/有害、助燃、 腐蚀、易燃、健康 危害、对 环境 有害。
  • 处理 前 先 把 标志 跟 意思 对 上。
🟥 红牌 · 常见错误 / Red Card
✘ WRONG 错
Corrosive and toxic mean the same thing.
腐蚀 跟 有毒 是 同 一 回事。
✔ RIGHT 对
Corrosive = attacks/destroys materials or skin on contact. Toxic = poisons the body if taken in. A substance can be one, the other, or both — they are different hazards.
腐蚀 = 接触 就 破坏 材料 或 皮肤。有毒 = 进入 身体 会 中毒。一 种 物质 可能 只 占 一 个, 也 可能 两 个 都 占 — 是 不同 的 危险。
⚖️ 考试 vs 真实 / Exam vs Reality
  • In the exam, write that the pH scale goes from 1 to 14.
  • In real life, very strong acids can have a pH below 0, and very strong bases above 14.
  • The 1-14 scale is a simplified range for everyday substances.
  • 考试 时 写: pH 是 1 到 14。
  • 真实 中, 很 强 的 酸 pH 可 到 0 以下, 很 强 的 碱 可 到 14 以上。
  • 1-14 只 是 给 日常 物质 用 的 简化 范围。
🔗 比喻 / Analogy
  • A science answer is like a receipt: without evidence, nobody trusts the purchase.
  • 科学答案像收据：没有证据，别人不信你买过。
🎯 考官要什么 / What the examiner wants
  • State = short answer.
  • Describe = say what and how.
  • Explain = say why (fact + reason).
  • Define = exact syllabus wording.
  • State = 简短作答。
  • Describe = 说出是什么、怎样。
  • Explain = 说出为什么(事实+原因)。
  • Define = 用大纲原话精确定义。
