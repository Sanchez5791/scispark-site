The pH Scale (0-14)

pH 标度 (0-14)
The pH scale runs from 0 to 14 · pH 标度从 0 到 14
  • The pH scale is a number line from 0 to 14.
  • 0 to 6 = acidic.
  • 7 = neutral.
  • 8 to 14 = alkaline.
  • Lower number = more acidic.
  • Higher number = more alkaline.
  • pH 标度是从 0 到 14 的数字线。
  • 0 到 6 = 酸性。
  • 7 = 中性。
  • 8 到 14 = 碱性。
  • 数字越小 = 越酸。
  • 数字越大 = 越碱。

Each pH has a colour · 每个 pH 都有一个颜色
  • Universal Indicator turns a different colour for each pH value.
  • Strong acid = red.
  • Weak acid = orange/yellow.
  • Neutral = green.
  • Weak alkali = blue.
  • Strong alkali = purple.
  • Match the colour to a chart to read the pH.

  • 中性 = 绿。
  • 弱碱 = 蓝。
  • 强碱 = 紫。
  • 把颜色对到色卡上就能读出 pH。

Where everyday liquids sit on the scale · 日常液体在标度上的位置
  • Lemon juice and vinegar are acidic (low pH).
  • Pure water is neutral (pH 7).
  • Soap and oven/paint cleaners are alkaline (high pH).
  • Use only Global Commons examples: water, lemon, vinegar, soap.
  • 柠檬汁和醋是酸性 (pH 低)。
  • 纯水是中性 (pH 7)。
  • 肥皂和清洁剂是碱性 (pH 高)。
  • 只用全球通用的例子: 水、柠檬、醋、肥皂。
🟥 红牌 · 常见错误 / Red Card
✘ WRONG 错
A higher pH number always means more dangerous.
pH 数字越大就越危险。
✔ RIGHT 对
A higher pH means more alkaline, NOT more dangerous. Both very low pH (strong acid) and very high pH (strong alkali) can be dangerous. pH 7 (neutral) is the safe middle.
pH 越大代表越碱, 不是越危险。 很低的 pH (强酸) 和很高的 pH (强碱) 都可能危险。 pH 7 (中性) 是安全的中间。
⚖️ 考试 vs 真实 / Exam vs Reality
  • 📝 考试: The pH scale goes from 0 to 14, with 7 = neutral.
  • 🔬 真实: Very strong acids can have a pH below 0, and very strong alkalis can have a pH above 14. The 0-14 scale is a simplified range for everyday substances, not a physical limit.
  • In the exam, write pH 0 to 14.
  • In real life the scale can go beyond both ends.
  • 📝 考试: pH 标度从 0 到 14, 7 = 中性。
  • 🔬 真实: 很强的酸 pH 可以低于 0, 很强的碱 pH 可以高于 14。 0-14 是日常物质的简化范围, 不是物理上限。
  • 考试时写 pH 0 到 14。
  • 真实生活里两端都可以超出。
🔗 比喻 / Analogy
  • The pH scale is a number ruler for sour-to-slippery behaviour.
  • pH 标度就是一把数字尺，用来量“酸到碱”的程度。
