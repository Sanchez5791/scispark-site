Hook · Real-world spark
Element / Compound / Mixture
元素 / 化合物 / 混合物
DouDou · Element / Compound / Mixture
Hook video · 60-90 sec · Coming soon
Element 元素 — e.g. pure copper. 元素 — 例如纯铜。
Compound 化合物 — e.g. water. 化合物 — 例如水。
Mixture 混合物 — e.g. air. 混合物 — 例如空气。

Everything around you is one of these three. 你身边所有东西都是这三种之一。

💡 Aside · 补充

The big question is simple: one atom type, joined atom types, or not joined? (If air could talk: "Don't call me a compound — I'm a messy group chat.") 核心问题很简单:一种原子?几种原子绑在一起?还是只是混在一起? (空气如果会讲话:「别叫我化合物,我是很乱的群聊。」)

Learn · Mechanism
Three kinds, one test
三种,一个判断
Element 元素 — only one type of atom; cannot be broken down chemically. 元素 — 只含一种原子;化学方法不能再分。
Compound 化合物 — two+ atom types chemically joined, e.g. H2O, NaCl. 化合物 — 两种以上原子化学结合,例如 H2O、NaCl。
Mixture 混合物 — two+ substances NOT joined; separable physically; no fixed proportion, e.g. air, seawater. 混合物 — 两种以上物质没结合;可物理分开;没有固定比例,例如空气、海水。
Read the diagram
看粒子图

Same-colour joined = element molecule; different-colour joined = compound; separate unjoined groups = mixture. 同色连一起 = 元素分子;不同色连一起 = 化合物;分开没连的几组 = 混合物。

🟥 Red Card · Common Mistake · 误区红牌

✗ Wrong: Calling a mixture a compound just because particles are close, or calling NaCl a mixture. 因为粒子挨在一起就把混合物当化合物,或把氯化钠当混合物。

✓ Right: Compound = atoms chemically JOINED (NaCl). Mixture = NOT joined, separable physically (seawater). 化合物 = 原子化学结合(NaCl)。混合物 = 没结合,可物理分开(海水)。

"Joined vs not joined" is the deciding test; getting it backwards loses the mark. 「有没有结合」是判断关键,搞反就不给分。

💡 Aside · 补充

Element = one kind of LEGO brick. Compound = different bricks clipped in a fixed build. Mixture = different builds in one box. (But chemical bonds are real forces, not plastic clips.) E-C-M: Element one kind, Compound connected, Mixture mixed. Close together ≠ compound. 元素=只有一种 LEGO。化合物=不同 LEGO 扣成固定作品。混合物=几种作品丢同一盒子。 (但化学键是真实作用力,不是塑料扣。)E-C-M 记法:Element 一种、Compound 化学连接、Mixture 只是混。挨得近≠化合物。

✦ Exam vs Reality · 考试 vs 真实

In the exam: Sort matter into element, compound or mixture. 考试分元素、化合物、混合物三类。

Real science: Alloys like steel count as mixtures, but their properties can look compound-like. 合金如钢算混合物,但性质有时像化合物。

What you learned today 今天学到了什么

Element = one type of atom. 元素 = 一种原子。
Compound = two+ atom types chemically joined. 化合物 = 两种以上原子化学结合。
Mixture = two+ substances not joined, separable physically. 混合物 = 两种以上物质没结合,可物理分开。
Diagrams: same-colour joined = element; different-colour joined = compound; separate = mixture. 粒子图:同色连=元素;不同色连=化合物;分开=混合物。
💡 Aside · 补充

One kind = element. Different joined = compound. Not joined = mixture. 一种=元素。不同原子连住=化合物。没连住=混合物。
