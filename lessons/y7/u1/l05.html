HOOK · Periodic Table (Intro)
元素周期表（入门）
HOOK
•
About 90 elements is a lot to remember.
大约 90 种元素,很难全记住。
•
Scientists put them in order in one big chart called the Periodic Table, so every element has its own place.
科学家把它们排进一张大表,叫周期表,每种元素都有自己的位置。
•
A computer chip, a copper wire, the oxygen you breathe — every one of these is an element with a place on the Periodic Table.
电脑芯片、铜线、你呼吸的氧气 —每一个都是周期表上有位置的元素。
•
Dmitri Mendeleev (Russia) arranged the first widely used Periodic Table.
CAREER ANCHOR (Around the World scientist):Dmitri Mendeleev(俄国)排出了第一张被广泛使用的周期表。
•
Today chemists and materials scientists use it to choose the right element for a job.
今天化学家和材料科学家用它来挑选合适的元素。

LEARN · Periodic Table (Intro)
元素周期表（入门）
LEARN
The Periodic Table lists all elements in order
•
The Periodic Table is a chart of all known elements, set out in order so each element has its own box.
周期表是一张列出所有已知元素的表,按顺序排好,每种元素一个格子。
IMAGE PENDING · L05_learn_01

Each box shows one element symbol
•
Each box in the table holds one element's chemical symbol, e.g.
表里每个格子放一种元素的化学符号,例如 Na 是钠、Si 是硅、H 是氢。
•
Na for sodium, Si for silicon, H for hydrogen.
Metals left/middle, non-metals right
•
Most metals sit on the left and middle of the table.
大多数金属在表的左边和中间。
•
Non-metals sit on the right.
非金属在右边。
•
Metals conduct electricity and heat, and are usually malleable and ductile.
金属导电、导热,通常可锤打、可拉成丝。
🔴 RED CARD · 误区红牌
✘
Sodium does not conduct heat" / treating a metal as having non-metal properties.
✔
Metals (e.g. sodium) conduct heat AND electricity, and are ductile and malleable. WHY IT picking the wrong property loses the mark. ZH 错: 「钠不导热」/ 把金属当成有非金属性质。 ZH 对: 金属(例如钠)既导电又导热,可拉成丝、可锤打。 ZH 为何重要: 选错性质就不给分。

Groups
•
A column is called a group; a row is called a period.
一列叫一个 group;一行叫一个 period。
•
Sodium is in Group 1.
钠在 Group 1。
•
This lesson only reads positions and symbols — no calculations.
这课只看位置和符号,不做计算。
✦ Exam vs Reality · 考试 vs 真实
•

"考试时分:金属 / 非金属。
•
In real life, there's a middle group called 'metalloids' — they act a bit like both.
真实中间还有 'metalloid'(类金属),像 silicon(硅)。
•
Silicon is the most famous one (it makes computer chips)."

📝 SUMMARY · 总结
•
The Periodic Table puts all elements in order.
周期表把所有元素排好顺序。
•
Each box shows one element symbol.
每个格子一个元素符号。
•
Metals are left/middle; non-metals are right; metalloids sit between.
金属在左中,非金属在右,类金属在中间。
•
Columns = groups; rows = periods.
列 = group;行 = period。
•
One element on its own is pure.
单独一种元素是纯的。
•
But what happens when elements join up, or just mix?
但元素结合在一起、或只是混在一起,会怎样? 下一课:元素、化合物、混合物。
