Learn · Define · 学习 · 定义

Organs → Organ Systems器官 → 器官系统

•
An organ system is a group of organs that work together to do one big job for the body.器官系统 = 一群器官一起工作,为身体做一件大工作。
•
Digestive system breaks down food. Circulatory system transports blood. Respiratory system does gas exchange. Nervous system controls and coordinates.消化系统分解食物。循环系统运输血液。呼吸系统做气体交换。 神经系统控制和协调。

📌 Summary · 总结
•
Level 4 of the hierarchy: several organs working together make an organ system. Examples: digestive, circulatory, respiratory, nervous, skeletal.层级第四层:好几个器官一起工作 = 器官系统。 例子:消化、循环、呼吸、神经、骨骼系统。
•
Organs teamed up into systems. But your digestive system, circulatory system and all the others must work together too. What do you call all the systems combined? Next: Organ Systems -> Organism.器官组成系统。但你的消化系统、循环系统等等也必须一起工作。 所有系统合起来叫什么?下一课:器官系统 -> 生物体。
