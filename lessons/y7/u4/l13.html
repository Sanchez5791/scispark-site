Organ Systems → Organism器官系统 → 生物体

A pandemic spreads (e.g. a flu or coronavirus outbreak). A virus enters one type of cell, but a whole person can become ill — every organ system is affected. An epidemiologist studies how this moves from cell to whole organism to whole population.大流行病传开（例如流感或冠状病毒）。 病毒先进入一种细胞，但整个人都会生病——所有器官系统都受影响。 流行病学家研究它怎么从一个细胞、到整个生物体、再到整个人群。
Learn · Define · 学习 · 定义

Organ Systems → Organism器官系统 → 生物体

•
An organism is built in five levels, smallest to largest: cell -> tissue -> organ -> organ system -> organism. Similar cells form a tissue. Different tissues form an organ. Organs that work together form an organ system. All the organ systems together form one whole organism.生物体由五层组成，从小到大： 细胞 -> 组织 -> 器官 -> 器官系统 -> 生物体。 相似的细胞组成组织。不同的组织组成器官。 一起工作的器官组成器官系统。所有器官系统合起来是一个完整生物体。

🟥 RED CARD · Common mistake🟥 红牌 · 常见错误
✘ The error✘ 错误想法
An organism is just a big pile of cells.生物体只是一大堆细胞。
✓ Correct idea✓ 正确想法
Wrong. The levels are organised, not piled. Cells are organised into tissues, tissues into organs, organs into systems, systems into one organism. Organisation is the point, not the number of cells.错。各层是有组织的，不是堆起来的。 细胞组成组织，组织组成器官，器官组成系统，系统组成一个生物体。 重点是“组织起来”，不是细胞数量。
🟫 Exam vs Reality🟫 考试 vs 真实
In the exam, the five levels are drawn as one clean ladder, cell -> tissue -> organ -> system -> organism, with one neat example per level. In real life the levels overlap and are not always tidy: blood is a tissue but it is liquid; some structures (like the skin) act as an organ that contains several tissue types; and not every cell sits inside an organ. The ladder is a model to organise ideas, not a perfect description of every body part.考试里五层画成一个干净的梯子， 细胞 -> 组织 -> 器官 -> 系统 -> 生物体，每层配一个整齐的例子。 真实里各层会重叠、不一定整齐：血液是组织，却是液体； 有些结构（例如皮肤）是含多种组织的器官； 也不是每个细胞都待在某个器官里面。 这个梯子是用来整理概念的模型，不是对每个身体部位的完美描述。
