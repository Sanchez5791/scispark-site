📝 SUMMARY · 总结
力有大小和方向,用箭头画。箭头越长,力越大。两边力相等就不动,不相等就动。
A force has size and direction, drawn as an arrow. Longer arrow = bigger force. Equal forces = no movement; unequal = it moves.
🧠 MEMORY HOOK · 记忆口诀
箭头越长力越大;两边相等就不动。
Longer arrow, bigger force; equal both sides, no move.
