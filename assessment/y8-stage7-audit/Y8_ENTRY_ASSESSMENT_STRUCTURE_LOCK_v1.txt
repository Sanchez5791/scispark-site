4. TOPIC BALANCE LOCK
==================================================

The Year 8 Entry Readiness Assessment must balance these readiness domains:

1. Biology / body systems / cells
2. Chemistry foundation
3. Ecosystem / living things
4. TWS / data / investigation
5. Forces
6. Primary gap-fill for magnetism, dissolving, separation, reversible/irreversible changes after patch

Important:
TWS is embedded, not isolated as a separate subject block.
Forces should not dominate.
Primary must not become the main assessment source.
Primary is used as targeted gap-fill.
